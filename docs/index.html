Let’s share it much quicker

Gisquick is an open source geospatial data publishing platform. We have developed it for rapid sharing of our projects created with QGIS. We put together minimal set of function for useful generic web mapping application. We developed it for ourselves as well as for you. Let’s use it and share our great GIS outputs with others easily and much quicker.

Learn more

QGIS Friendly

Gisquick is set of the mapping applications for publishing QGIS projects. It consists of plugin for QGIS, Django based server application, QGIS-server and web and mobile clients. The process of data publishing with Gisquick is quite easy. You can create and set up your projects with QGIS (layers, metadata, print layouts), prepare it for publishing using Gisquick plugin and upload it to the server. Use them together.

Open for challenges

Gisquick is developed as an open source project under GNU/GPL license. For client-server communication it uses OGC WMS, WMTS and WFS services. Gisquick is the solution without any extra costs (in comparison with common proprietary solutions). It’s applicable in everyday practice (e.g. education or nature conservancy) or for further custom development. Take full advantage of the Open source.

Easy to Deploy

All Gisquick server components can be easily deployed in Docker containers. You can start serving your GIS projects from the cloud (like Amazon Web Services) or from your own hardware without any troubles. For project creators it's even simpler, all they need is QGIS desktop software with Gisquick plugin installed. Try it yourself or use our expert help. Contact us.

Practical Anywhere

Gisquick web client gives you an easy to use user interface for viewing your GIS projects online. It’s designed as responsive and mobile-friendly application, but a dedicated mobile client (Cordova hybrid app) for even better experience is also available. Use it as you need to solve your tasks anywhere.




Suited to anyone

Everything you need for creating maps in Gisquick is located at one place - on the side panel. It offers ready to use layer compilations (topics) suitable for basic users and well arranged layers (base and overlay layers linking up with attribute data tables and layer metadata) for users who are used to working with common GIS. Use Gisquick to more effective work with your end users.



Linked to tables

Working with data tables is very intuitive and quick. Gisquick combines map and table view on your datasets. You can enjoy map features (paning, zooming, navigating or geolocating) and features of database table (searching, filtering accross big data tables or zooming and higlighting objects from table on the map) together in one application and at the same time.

See what you print

Gisquick web client uses innovative print tool with advanced layout preview. Map will be automatically scaled to display whole page in selected zoom level, regardless of screen resolution. You can create a custom print templates in QGIS desktop and use them in Gisquick web application.

Automatic tile caching

Gisquick server support server-side map tiles caching without any additional configuration. You can simply enable it for your projects with less often changing geometry data and achieve better performance and faster access to your maps.
