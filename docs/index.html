Let’s share GIS much quicker

Gisquick is an open source geospatial data publishing platform. We have developed it for rapid sharing of our projects created with QGIS. We put together minimal set of function for useful generic web mapping application. We developed it for ourselves as well as for you. Just use it to solve today’s problems and show the best solution for tomorrow.

QGIS

Gisquick is set of the mapping applications for publishing QGIS projects. It consists of plugin for QGIS, Django based server application, QGIS-server and web and mobile clients.

The process of data publishing with Gisquck is quite easy. You can create and set up your projects with QGIS (layers, metadata, print layouts), prepare it for publishing using Gisquick plugin and upload it to the server.

Open solutions

Gisquick is developed as an open source project under GNU/GPL license. For client-server communication it uses OGC WMS, WMTS and WFS services. Gisquick is the solution without any extra costs (in comparison with common proprietary solutions). It’s applicable in everyday practice (e.g. education or nature conservancy) or for further custom development.

Web and mobile

Gisquick web client gives you an easy to use user interface for viewing your GIS projects online. It’s designed as responsive and mobile-friendly application, but a dedicated mobile client (Cordova hybrid app) for even better experience is also available.

Deployment

All Gisquick server components can be easily deployed in Docker containers. You can start serving your GIS projects from the cloud (like Amazon Web Services) or from your own hardware without any troubles. For project creators it's even simpler, all they need is QGIS with Gisquick plugin installed.

Map content

Gisquick web client has clean user interface. Everything you need for creating maps is located at one place - on the side panel. It offers ready to use layer compilations (topics) suitable for basic users and well arranged layers (base and overlay layers linking up with attribute data tables and layer metadata) for users who are used to working with common GIS. It also contains a map legend.

Data tables

Working with data tables is very intuitive and quick. Gisquick combines map and table view on your datasets. You can enjoy map features (paning, zooming, navigating or geolocating) and features of database table (searching, filtering accross big data tables or zooming and higlighting objects from table on the map) together in one application and at the same time.

Print

Gisquick web client uses innovative print tool with advanced layout preview. Map will be automaticaly scaled to display whole page in selected zoom level, regardless of screen resolution.

With Gisquick, you can create a custom print templates in QGIS desktop and use them in web application.

Automatic server-side tile caching

Gisquick server support map tiles caching without any additional configuration. You can simply enable it for your projects with less often changing geometry data and achieve better performance and faster access to your maps.
